NVE Molecular Dynamics

Overview

The NVE ensemble (microcanonical ensemble) conserves particle number (N), volume (V), and total energy (E). It is the most fundamental MD ensemble: no thermostat or barostat is applied, so the system evolves under pure Newtonian mechanics driven by the ML potential forces.

NVE is used primarily as a production ensemble after a system has been pre-equilibrated under NVT conditions. Because there is no artificial coupling to an external bath, dynamical properties derived from NVE trajectories—such as diffusion coefficients, velocity autocorrelation functions, and vibrational spectra—are unperturbed by thermostat artifacts.

Energy conservation quality is automatically assessed at the end of each run. MAPLE reports the linear drift rate, relative energy fluctuation, and kinetic-potential energy anti-correlation as standard quality metrics.

Parameters

Parameter Type Default Description
timestep float 0.25 Integration time step in femtoseconds. The default (0.25 fs) is conservative and recommended for ML potentials to ensure energy conservation.
steps int 400000 Total number of integration steps (= 100 ps at 0.25 fs default).
temperature float 300.0 Temperature in Kelvin used only for initializing velocities from the Maxwell–Boltzmann distribution. Has no effect when init_from is set.
traj_every int 100 Write a trajectory frame every N steps.
log_every int 100 Write a thermodynamic log entry every N steps.
remove_com_every int 100 Remove center-of-mass translational momentum every N steps to prevent slow drift from floating-point accumulation.
remove_rotation bool False Also remove angular momentum about the center of mass. Recommended only for isolated gas-phase molecules where overall rotation is not of interest.
init_velocities bool True Initialize velocities from a Maxwell–Boltzmann distribution. Set to False when loading velocities from a prior trajectory via init_from.
init_from str None Path to a *_md_final.xyz file. Loads positions and velocities to continue from a prior run (e.g., after NVT equilibration).
resume bool False Resume an interrupted simulation, appending to existing trajectory and log files and offsetting the step counter.
random_seed int None Integer seed for reproducible velocity initialization. If None, a random seed is used.
verbose int 1 Output verbosity (0 = energy only, 1 = standard thermodynamic output).

Input Example

NVE production run starting from an NVT-equilibrated structure:

#model=ani2x
#md(ensemble=nve, steps=400000, init_from=water_nvt_md_final.xyz)
#device=cpu

XYZ water_opt_opt.xyz

Simple NVE run from scratch with fresh velocity initialization:

#model=ani2x
#md(ensemble=nve, temperature=300, steps=200000, timestep=0.25)
#device=cpu

O    0.000000    0.000000    0.117300
H    0.000000    0.757200   -0.469200
H    0.000000   -0.757200   -0.469200

Energy Conservation

After the simulation completes, MAPLE prints a summary of energy conservation quality. The reported metrics are:

  • Linear drift rate: slope of a linear fit to the total energy over the second half of the trajectory (Eh/ps). A well-behaved NVE run should show drift < 10−4 Eh/ps for typical molecular systems.
  • Relative energy fluctuation: σ(E)/|⟨E⟩|, the standard deviation of the total energy relative to its mean. Values below 10−4 indicate excellent conservation.
  • KE–PE anti-correlation: for a well-conserved NVE trajectory this approaches −1.0, reflecting the compensation between kinetic and potential energy fluctuations.

If energy conservation is poor, reduce the timestep or verify that the input structure was properly equilibrated at the target energy before starting the NVE run.

Tip

Always pre-equilibrate with NVT before running NVE production. Starting NVE from a poorly relaxed structure or from an arbitrary velocity distribution leads to large initial energy fluctuations and unreliable statistics.
